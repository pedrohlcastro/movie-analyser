Dados dos Filmes

Uso API The Movie DB para obter imagens e informações sobre os filmes. Assim que a busca é feita no site a API é ativada usando a base do The Movie DB.

Após isso, é feito a apresentação dos resultados usando Angular 4 e Angular Material. Dando a opção do usuário navegar para ver mais informações sobre o filme clicando no botão do Card.

Na parte de mais informações o usuário pode ver uma Sinopse do filme e ver as analises de emoção dos reviews feitos.

 Review The New York Times

Uso API The New York Times para obter review do filme, além disso foi preciso fazer scrapping na página do Jornal.

A API do Jornal retorna a URL da review que é passada para o Back end, onde ocorre o scrapping retirando somente o texto do URL, ficando assim um text/plain. Que será analisado pelo Watson e retornado para o front end.

 IBM Watson Tone Analizer

Uso do IBM Watson para fazer análise de emoções dentro do texto de review do Jornal.

As emoções são recebidas pelo front end, onde ocorre a tradução e onde são setados as cores delas para serem exibidas para o usuário em forma de gráficos de pizza.
