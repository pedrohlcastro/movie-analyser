Dados dos Filmes

É utilizado a API: "The Movie DB" para obter imagens e informações sobre os filmes. Assim que a busca é feita no site a API é ativada usando a base do The Movie DB.

Após isso, é feito a apresentação dos resultados usando Angular 4 e Angular Material. Dando a opção do usuário navegar para ver mais informações sobre o filme clicando no botão do Card.

Na parte de mais informações o usuário pode ver uma Sinopse do filme, além da analise de emoções da critica feita no Jornal The New York Times.

 Review The New York Times

A API do jornal The New York Times é utilizada para obter review do filme, além disso foi preciso fazer scraping na página do Jornal.

A API do Jornal retorna a URL da review que é passada para o Back end, onde ocorre o scraping retirando somente o texto do URL, ficando assim um text/plain. Que será analisado pelo Watson e retornado para o front end.

 IBM Watson Tone Analizer

IBM Watson é usado para fazer análise de emoções dentro do texto de review do Jornal.

As emoções são recebidas pelo front end, onde ocorre a tradução e onde são setados as cores delas para serem exibidas para o usuário em forma de gráficos de pizza.
